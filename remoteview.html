El Vínculo Anacrónico

Modo Precognición

El sistema te muestra 3 cartas catalizadoras al azar como disparador. Eliges la carta que crees que el sistema generará a continuación. Solo después de pulsar Submit el QRNG cuántico selecciona la carta Target. Pones a prueba la precognición: percibir algo que aún no existe.

Modo Clarividencia

El sistema selecciona el Target antes de empezar la ronda y lo coloca boca abajo. Las 3 cartas catalizadoras sirven para sintonizar con la identidad ya determinada del Target. Cuando pulsas Submit, el Target se voltea. Pones a prueba la clarividencia: percibir algo que ya existe pero está oculto.

Recargar el Trigger

Si las 3 cartas catalizadoras no te resuenan, haz doble tap sobre ellas para sortear una nueva triada. Solo es posible antes de pulsar Submit — una vez registrada la predicción, el trigger queda fijado para mantener limpio el log estadístico.

Aleatoriedad Cuántica

Las cartas se seleccionan con el mismo generador cuántico de Radio Arkana — bytes obtenidos de fluctuaciones del vacío. Si el pool se agota, se usa Math.random() como respaldo (el indicador en la barra superior pasa a ámbar).

Lectura del P-Value

P-Value < 0.05 sugiere un resultado significativamente distinto del azar (1/22 ≈ 4.5%). No es prueba de fenómeno psi — solo significa que el patrón es difícil de explicar por suerte sola. Necesitas muchos rounds para que el test tenga poder estadístico.
